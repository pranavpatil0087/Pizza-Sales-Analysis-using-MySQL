# 🍕 Pizza Sales Analysis using MySQL

## 📌 Overview

This project analyzes Pizza Hut sales data using **MySQL** to uncover valuable business insights. It demonstrates SQL skills ranging from basic queries to advanced analytical techniques.

## 🛠 Tech Stack

* MySQL
* SQL
* MySQL Workbench

## 📊 Key Analysis

* Total Revenue
* Highest Priced Pizza
* Most Ordered Pizza Size
* Top 5 Best-Selling Pizzas
* Category-wise Sales Analysis
* Hourly Order Distribution
* Average Daily Orders
* Monthly Revenue Trend
* Revenue Contribution by Category
* Cumulative Revenue
* Top Revenue-Generating Pizzas
* Average Order Value

## 💡 SQL Concepts Used

* Joins
* Aggregate Functions (`SUM`, `AVG`, `COUNT`)
* `GROUP BY`, `ORDER BY`
* Subqueries
* Window Functions (`RANK()`, `SUM() OVER()`)
* Date & Time Functions

## 📂 Dataset

* `orders`
* `order_details`
* `pizzas`
* `pizza_types`

## 🚀 How to Run

1. Create the `pizzahut` database.
2. Import the CSV files.
3. Run the SQL script.
4. Execute the queries to explore business insights.
